# Hardihood
Status monitoring & alerting, done simply. Create status-checks using Lua script,
which get displayed on a simple, clear monitor.


## Development
### tl;dr
```bash
$ git clone git@github.com:clagraff/hardihood.git
$ go build
$ ./main
```
